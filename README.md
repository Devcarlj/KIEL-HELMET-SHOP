# Kiel Helmet Shop: E-Commerce Platform

Kiel Helmet Shop is a streamlined motorcycle protective gear e-commerce platform that transitions manual social media ordering into an automated, full-stack digital storefront. It efficiently manages product catalogs, user shopping carts, and secure checkout operations within a modern, high-contrast interface.

## Key Features
* **Dynamic Digital Storefront:** Replaces manual Facebook-based ordering with a responsive, high-contrast catalog featuring premium helmets, jackets, and essential riding armor.
* **Optimized Cart & Checkout Logic:** A state-managed shopping cart system handling real-time price updates, quantity changes, and secure checkout workflows.
* **Inventory Management System:** A robust backend system allowing owners to catalog, monitor stock levels, and update product listings instantly.
* **Virtual Fitment Expert:** Integrated AI capabilities powered by the Gemini API to assist riders in finding the exact gear sizes and specifications based on their personal needs.

## Tech Stack
* **Frontend:** React.js, Tailwind CSS
* **State Management:** Redux Toolkit
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose ODM)
* **AI Integration:** Google Gemini API
* **Deployment:** Vercel
